# Frontend

Next.js frontend for the EHR Query Engine.

## Setup

See main [README.md](../README.md) for full setup instructions.

### Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Project Structure

See main [README.md](../README.md) for full project structure.

## Components

- **ChatInterface**: Natural language query input
- **DatasetExplorer**: Dataset preview and metadata
- **DatasetExport**: Export interface with format selection
- **ProtectedRoute**: Authentication wrapper
